Jobs Board Skeleton

This is a skeleton example of a jobs board application. Please note that there is absolutely no UI design and can be made as aesthetically pleasing as you'd like with html5. I am powered by Django and written in Python. Currently I am only using the careerbuilder.com API to find jobs. More APIs or even web scraping can be easily implemented.

Currently I am only using the Location and Keywords parameters when searching for jobs. A complete list of parameters is located here: http://api.careerbuilder.com/JobSearchInfo.aspx
